Question-1: What are the different ways to select an element in the DOM?

There are many way's to select an element in the DOM.Like by Id : [document.getElementById("id")], by class name : [document.getElementByClassesNames("className")], by tag name : [document.getElementByTagName("tagName")],using query selectors : [document.getElementByquerySelector("CSS selector"), to get multiple selectors : -- [document.getElementByquerySelectorAll("CSS selector")]] ,and by attibute name : [document.getElementByName("nameAttribute")].

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML : It represents the HTML markup inside an element.Both HTML tags and content can be set here. innterText : It represents the text inside an element. Displaying or modifying the text. textContent : It also represents the text content of an element.

Question-3: What is event delegation in the DOM?

Event delegation is a way , where attaching a single event listener to the parent element we can handle the child elements. No need to add event listener repeatedly on child elements.

Question-4: What is event bubbling in the DOM?

Event Bubbling : it's a way where, by adding event on a child element it allows parent elements to detect and respond to events that occur on their child elements. After trggering, the event Bubbles up to the parent element until it reaches hte document leve.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element we need to use the document.createElement()method. then we need to add the context inside it by newElement.textContent. Then if we need to add the element we need to use document.body.appendChild(newElement)method. And if we need to remove the element , we need to use the parent.removeChild(child)method.
